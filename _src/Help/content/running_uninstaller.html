Once you have launched the Uninstaller, you are prompted to follow the instructions on the screens:

Step 1 – Review Program Location

Review the location from which KO Approach is being removed and click Next to continue.

Step 2 – Remove User-specific Folders

The Uninstaller attempts to find all KO Approach directories that have been created for the various users of your computer. This information is retrieved from the Windows Registry and will not include the directories that have been created for those users whose access rights do not allow writing to the Registry. Check the box if you wish these items to be removed and click Next to continue.

ATTENTION! If you chose to remove the directories listed on the screen, please make sure that there is no important data stored in them as this data will be deleted! Unless you are positive that you want all items removed, leave the box unchecked.

Step 3 – Exit KO Approach

Quit the the running instance of KO Approach by right-clicking the Tray icon and choosing Exit from the menu, then click Retry to proceed with the uninstallation.

Note: This step only needs to be performed if an instance of Approach is currently running.

Step 4 – Finish

You can optionally click the Show Details button to see which files are being removed from your system. Once the process is completed, the Uninstaller will display the Uninstallation Complete message. Click Finish to exit the Uninstaller.
