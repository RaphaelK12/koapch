KO Approach is shareware, which means that you must either purchase it or permanently remove it from you computer. KO Approach can run in two modes:
  1. Unregistered mode;
  2. Registered mode.
The Unregistered mode is only used for trial purposes. You may not use Approach in this mode longer than 30 days. Before you make a purchase some of the Approach functionality is limited:
  • only four menus can be displayed at a time;
  • only two main features can be turned on at a time;
  • only two plugins can be loaded.
In order to unlock the full functionality, you must activate your copy of Approach. The process consists of three steps:
  1. Purchasing KO Approach on our Order Page ;
  2. Receiving an e-mail from our Registration service, containing your Serial Number;
  3. Supplying the received Serial Number to the Activation Service and obtaining an Activation Key. This is done on the Registration Tab.

Choosing a license

Four types of licenses are offered, differing in price and in the number of computers from which the Activation Key can be requested.

Please note that each Activation Key is only valid for each particular computer.This is done by collecting certain hardware parameters from your machine. Our Registration Service keeps track of your requests for Registration Keys, including the collected hardware parameters.
Within the number of requests for Registration Keys, no limit is set on the number of computers on which you may simultaneously install and run Approach. However, if our records indicate that you have already exceeded the limit of Key requests, no Key will be sent and you will have to purchase another license.
Also, you can request an Activation Key on the same machine unlimited number of times. Hence, if you have accidentally lost your Activation Key, you can request it again with no risk.

Before you make a purchase

Even a minor hardware upgrade can result in an invalid Activation Key, which will require that you make an additional request. If you plan to make frequent hardware upgrades to your computer, consider ordering a License that will allow convenient usage without having to spend money several times. Also, if you plan to install Approach on multiple computers, choose a license that allows an appropriate number of requests for Activation Key.
