The following files and folders are placed on your hard drive during the installation of KO Approach:

File/Folder Name Default Installation Location Description
Approach.chm <Program Files>\KO Approach This help file
Approach.exe <Program Files>\KO Approach Actual KO Approach executable
Approach.key <Program Files>\KO Approach KO Approach Activation Key. This file is only present if you have activated your copy of KO Approach
ApproachIpc.dll <Program Files>\KO Approach Helper DLL which enables the Folder Menus and Titlebar Menus features
ClipboardAssistant.dll <Program Files>\KO Approach\Plugins ClipboardAssistant plug-in module
Locale <Program Files>\KO Approach Folder for storing additional language files
InstantTxt.dll <Program Files>\KO Approach\Plugins InstantTxt plug-in module
InstantWave.dll <Program Files>\KO Approach\Plugins InstantWave plug-in module
License.txt <Program Files>\KO Approach End-User license agreement
readme_scope.txt <Program Files>\KO Approach\Plugins Scope plug-in description and build history
Scope.dll <Program Files>\KO Approach\Plugins Scope plug-in module
Uninstall.exe <Program Files>\KO Approach KO Approach Uninstaller program
Uninstall KO Approach.lnk <Start Menu - All Users>\Programs\KO Approach KO Approach Uninstaller shortcut
KO Approach.lnk <Start Menu - All Users>\Programs\KO Approach Shortcut to launch KO Approach
KO Approach Help.lnk <Start Menu - All Users>\Programs\KO Approach Shortcut to launch this help file
KO Approach.lnk <Start Menu - Current User>\Programs\Startup Program auto-run shortcut. KO Approach automatically creates/removes this shortcut whenever you toggle the Enable auto-launch checkbox on the Maintenance tab

Note: Some files may not be present on your hard drive depending on the choices you made during the installation.
