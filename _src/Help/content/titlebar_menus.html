The Titlebar Menus feature displays a menu listing all the items that constitute the path to the current Window. The items appear in the hierarchical order, with the parent positioned immediately above the child and the Desktop occurring on the top.

To activate this menu, click anywhere on the Title bar of any open Explorer Window while holding the CTRL key depressed.

You can switch to the inverse folder ordering mode by modifying the Folder order setting on the Features tab of the Options window. On this tab you can also change additional parameters that affect the way Titlebar Menus displays folder hierarchy.
