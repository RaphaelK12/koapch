Q: I am using the single-click mode to open folders. Can I take advantage of using KO Approach?
A: Yes, you can. Simply place your mouse cursor over a folder and wait for a few moments leaving the pointer inactive. The menu will appear shortly.
Q: I am trying to click a folder and drag it somewhere but the menus prevent me from being able to do so.
A: The Initial Menu Delay is probably set to a very small value. Try increasing this value to allow yourself more time for grabbing the folder.
Q: I click a folder and wait long enough, but no menu appears. What's wrong?
A: Make sure that Folder Menus option is enabled. The checkbox control for this option is located on the Features tab. Also, make sure that you installed an appropriate edition of KO Approach: a 32-bit edition should only be installed on a 32-bit Operating System, and 64-bit edition on a 64-bit Operating System.
Q: I am a non-English-speaking user. Is KO Approach available in my native language?
A: KO Approach can be localized into any language. Please refer to our Web site for additional details.
Q: I am a software developer. Can I write my own plug-ins for KO Approach?
A: KO Approach architecture is still being tested, so third-party plug-ins are currently not supported. However, we do plan to release an SDK in the future (hopefully, with the release of version 1.0) to enable developers to create their own plug-in modules.
