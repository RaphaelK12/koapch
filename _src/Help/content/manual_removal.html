In rare instances, the Uninstaller may fail to remove some or all components of KO Approach from your hard drive, so the program will need be uninstalled manually.

Deleting the Files

  1. Exit KO Approach by clicking the Tray icon and selecting Exit from the menu.

  2. Delete the entire directory in which KO Approach was installed, typically, C:\Program Files\KO Approach.

  3. Delete all other files and folders that were placed on your hard drive during the installation of KO Approach. For the complete list of files, please refer to the Files Installed on Your Hard Disk section.

Deleting the Registry Key

  1. Click Start > Run.

  2. Type regedit and click OK.

  3. Locate the key HKEY_LOCAL_MACHINE\Software\KO Software\Approach and delete it.

  4. Locate the key HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Uninstall\KO Approach and delete it.

ATTENTION! This procedure involves modifying the Windows registry and should be performed by experienced users only. Modifications to the Windows registry, if not done correctly, can cause serious problems to a Windows system.

Deleting User-Specific Folders (optional)

  1. Run a search for KO Approach Items on your computer;

  2. The system typically finds entries similar to C:\Documents and Settings\[User Name]\KO Approach Items on Windows XP and older and C:\Users\[User Name]\Data\AppRoaming\KO Approach Items on Windows Vista and newer operating systems;

  3. Delete each of the found entries.
