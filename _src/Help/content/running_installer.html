Once you launch the Installer, you are given detailed instructions on each screen (actual screens may differ slightly from these examples):

Step 1 – Agree to License Terms

You must agree to the License Terms. Press the I Agree button to continue.

Step 2 – Select Program Components to Be Installed

  1. Component selection list

    Allows for selecting components that should be installed. The options are as follows:

    • Program Files – installs KO Approach core files. This component is required and cannot be unchecked.
    • Program Group – creates the KO Approach folder in the Programs section of the current user's Start Menu.
    • Auto-run – installs a shortcut to KO Approach into the Startup section of the current user's Start Menu. This option is identical to setting the Automatically launch Approach when OS Starts checkbox on the Maintenance tab.
    • Scope plugin – installs the Scope plug-in allowing you to preview graphics files from KO Approach menus.
    • InstantWave plugin – installs the InstantWave plug-in allowing you to play back WAV sound files from KO Approach menus.
    • InstantTxt plugin – installs the InstantTxt plug-in allowing you to read text files from KO Approach menus.
    • ClipboardAssistant plugin – installs the ClipboardAssistant plug-in which extends shortcut and folder contextual menus.

You can customize your installation of KO Approach by choosing only the components that you want. A description for each item will appear on the right once you highlight a component. After selecting desired components, click Next to proceed.

Step 3 – Specify Install Location

You can install KO Approach to the default location or choose your own. If the location you have entered does not exist, the Installer will create the necessary folder(s). Specify the desired location and click Next to continue.

Step 4 – Exit KO Approach

Quit the the running instance of KO Approach by right-clicking the Tray icon and choosing Exit from the menu, then click Retry to proceed with the installation.

Note: This step only needs to be performed if an instance of Approach is currently running.

Step 5 – Finish

Leave the Launch KO Approach checkbox checked if you want the progam to run immediately. If you opt to uncheck it, you can always start Approach manually at a later time. Click the Finish button to complete the Setup.

Completing the installation with the View Quick Start Guide checkbox checked opens a new window displaying the Quick Start Guide which provides an overview of the product's key features.

The Setup can automatically take you to the KO Approach License Ordering page if the Purchase a License for KO Approach checkbox is checked. If you are installing on top of an existing copy that already has a License Key, this checkbox will not be visible.
